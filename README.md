## ğŸ¯ Customization

### Adding Your Own Images
1. Place your product images in the `images/` folder
2. Update the CSS selectors (`.product-image`, `.hero-placeholder`, etc.) to use your images
3. Add `background-image` properties with your image paths

### Changing Colors
The main orange colors are defined in the CSS:
- Primary Orange: `#ff6b35`
- Light Orange: `#ff8c42`
- Dark Orange: `#ff5722`

Replace these values in `css/style.css` to change the color scheme.

### Adding Products
Update the product cards in `index.html` with your actual product information:
- Product names
- Descriptions
- Prices
- Images
